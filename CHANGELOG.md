# Changelog

All notable changes to this project are documented here.

This project follows Keep a Changelog and uses Semantic Versioning.

## [Unreleased]

### Added

- Release automation with release-plz and cargo-dist.

## [1.0.0] - 2026-06-08

### Added

- Initial stable ccplan CLI for TOML day plans.
- Native scheduler integration on Linux, macOS, and Windows.
- Agent-facing JSON reads, deterministic exit codes, and whole-plan stdin authoring.
- Shell completions and a generated `ccplan(1)` man page.
